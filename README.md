## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mindglow.git
cd mindglow
```

2. Install Firebase CLI (if not already installed):
```bash
npm install -g firebase-tools
```

3. Login to Firebase:
```bash
firebase login
```

4. Initialize Firebase project:
```bash
firebase init
```

5. Deploy to Firebase:
```bash
firebase deploy
```

## 🎯 Usage

Open `index.html` in your browser or visit the deployed Firebase hosting URL.

## 🔧 Configuration

### Firebase Setup
Update the Firebase configuration in `app.js` with your project credentials:
```javascript
const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-auth-domain",
  projectId: "your-project-id",
  storageBucket: "your-storage-bucket",
  messagingSenderId: "your-messaging-sender-id",
  appId: "your-app-id"
};
```
